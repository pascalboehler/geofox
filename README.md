# Geofox API Wrapper crate

This crate wraps the HVV / Hochbahn Geofox API for use in Rust Projects.

To get started, copy the `example.env` file to `.env` and add your credentials.

Currently, only **API Version 63** is supported 

The API credentials must be ordered via the official hvv channels.

**CURRENTLY UNDER DEVELOPMENT - DO NOT USE IN PRODUCTION**
